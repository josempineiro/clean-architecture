# Repository\<Entity\> Interface Documentation

Represents a generic interface for a repository, providing standard CRUD (Create, Read, Update, Delete) operations for entities of a specific type.

## Methods

### `getById(id: string): Promise<TEntity>`

Retrieves an entity by its unique identifier.

#### Parameters

- `id`: A string representing the unique identifier of the entity to retrieve.

#### Returns

A Promise that resolves to the retrieved entity of type `TEntity`.

### `getAll(): Promise<TEntity[]>`

Retrieves all entities of the specified type.

#### Returns

A Promise that resolves to an array of entities of type `TEntity`.

### `create(entity: TEntity): Promise<TEntity>`

Creates a new entity.

#### Parameters

- `entity`: An object representing the entity to be created.

#### Returns

A Promise that resolves to the created entity of type `TEntity`.

### `updateById(id: string, entity: TEntity): Promise<TEntity>`

Updates an existing entity with the provided data.

#### Parameters

- `id`: A string representing the unique identifier of the entity to update.
- `entity`: An object representing the updated data for the entity.

#### Returns

A Promise that resolves to the updated entity of type `TEntity`.

### `deleteById(id: string): Promise<void>`

Deletes an entity by its unique identifier.

#### Parameters

- `id`: A string representing the unique identifier of the entity to delete.

#### Returns

A Promise that resolves to `void` indicating successful deletion.
